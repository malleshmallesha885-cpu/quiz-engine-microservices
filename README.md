# Quiz Engine Microservices

Quiz Engine Microservices is a backend quiz platform built using Spring Boot and microservice architecture. It includes independent services for quiz, questions, users, and results, connected via REST APIs, Service Registry, and API Gateway.

## Tech Stack
- Spring Boot
- Microservices
- MySQL / H2
- Eureka Server
- Spring Cloud Gateway
- Feign Client
- Maven

## Services
- API Gateway
- Service Registry
- Question Service
- Quiz Service
- User Service
- Result Service

## Features
- Quiz creation and management
- Question categorization
- Distributed service communication
- Scalable backend design

## Purpose
This project is built for learning and demonstrating real-world microservice architecture using Spring Boot.
